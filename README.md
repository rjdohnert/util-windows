# util-windows
Util-windows is a complimentary package to Microsofts Coreutils that they  have just released at Build 2026.  It contains clones of system management tools you find in BSD and in Linux.  These tools are written in C++ and have  been tested on Windows 11, Windows Server 2022 and Windows Server 2025.  They  have been tested with Powershell and cmd.
