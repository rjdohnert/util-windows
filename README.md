Util-windows is a complimentary package to Microsofts Coreutils that they 
have just released at Build 2026.  It contains a number of basic system tools similar
to those found in VMS, UNIX and Linux.  These tools are written in C++ and have 
been tested on Windows 11, Windows Server 2022 and Windows Server 2025.  They 
have been designed to work with Powershell and using the Powershell command schema 
but they have been tested with both the current Powershell 7 and cmd. 
List of commands is listed in the release folder

Requirements
-------------
get-admin requires Powershell 7 to launch an Administrator Powershell consoile

Install
-------

Copy all commands from the releases folder into C:\Windows\system32 folder
